יש אפשרות בלחיצה על לינק הצג לראות פרטים נוספים 
יש אפשרות מחיקת חבר ואפשרות הוספת חבר חדש
בלחיצה על הצג יופיע לנו פרטים נוספים על החבר ואפשרות לערוך את כל הפרטים
